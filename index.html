[ About Me ]

Hello, I’m Saurav, an active Bug Bounty Hunter and Penetration Tester. I spend my days and nights hunting for vulnerabilities, contributing to a safer internet by reporting security flaws to organizations worldwide.

My core expertise lies in IDOR (Insecure Direct Object Reference) and API Security and Ai Security , where I’ve developed strong skills in identifying broken authorization flows and hidden API risks that attackers could exploit.
